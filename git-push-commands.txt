# Git Commands to Push to GitHub
# Run these in Cursor's integrated terminal (Ctrl+` to open terminal)

# 1. Check status
git status

# 2. Stage all files
git add .

# 3. Commit changes
git commit -m "Initial commit - Life-Lag project"

# 4. Set default branch to main (if needed)
git branch -M main

# 5. Push to GitHub
git push -u origin main
